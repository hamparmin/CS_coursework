Radial Basis Functions¶

In this section we are turning the pair of time inputs into a much richer representation, mapping (d,t) into ((d), (t)). In order to find out what parameters: $\sigma, \alpha $ and the number of RBF functions to use for this mapping, I experiment with different parameter values on just the training data. This can give us some estimates about good parameter values.

'''
Create functions to:

1. Split data at a specified cutoff point.
2. Apply the rbf transform on data given parameter values
3. Fit a Ridge Regression model on the data and evaluate model by MSE
4. Create a center array for x_days based on n_rbf - range(0,365)
5. Create a center array for x_minutes based on n_rbf - range (0,1435)

Index 525887 stands for the split between 2015 and 2016.

'''
def split_data(df,cutoff):
    x_train=[df.days.values[:cutoff],df.minutes.values[:cutoff]]
    x_test=[df.days.values[cutoff:],df.minutes.values[cutoff:]]
    y_train=df.temp.values[:cutoff]
    y_test=df.temp.values[cutoff:]
    return (x_train,x_test,y_train,y_test)

def rbf_transform(df,cutoff,center1,center2,sigma1,sigma2,alpha):
    data=split_data(df,cutoff)
    x1=np.array(rbf_kernel(data[0][0].reshape(-1,1),center1,gamma=1/sigma1))
    x2=np.array(rbf_kernel(data[0][1].reshape(-1,1),center2,gamma=1/sigma2))
    x_train=np.concatenate((x1,x2),axis=1)
    
    x3=np.array(rbf_kernel(data[1][0].reshape(-1,1),center1,gamma=1/sigma1))
    x4=np.array(rbf_kernel(data[1][1].reshape(-1,1),center2,gamma=1/sigma2))
    x_test=np.concatenate((x3,x4),axis=1)
    
    return x_train,x_test,data[2],data[3]


def regr(df,cutoff,center1,center2,sigma1,sigma2,alpha):
    x_train,x_test,y_train,y_test=rbf_transform(df,cutoff,center1,center2,sigma1,sigma2,alpha)
    regr_rbf=linear_model.Ridge(alpha=alpha,fit_intercept=True) #model selection
    regr_rbf.fit(x_train,y_train) #model fitting
    y_pred=regr_rbf.predict(x_train) #model prediction
    MSE=metrics.mean_squared_error(y_train, y_pred) #calculate MSE train for model
    return MSE

def center1(n_rbf):
    #function to allocate centers based on different number of RBFs to be specified
    return np.linspace(0,365,n_rbf).reshape(-1,1)

def center2(n_rbf):
    #function to allocate centers based on different number of RBFs to be specified
    return np.linspace(0,1435,n_rbf).reshape(-1,1)

'''
Estimate reasonable RBF parameter values by:
1. Creating RBF transforms with different: alphas, sigmas and n_rbf
2. Fitting a regression onto the rbf transforms.
3. Calculating and recording MSE (train MSE) for each regression.
4. Use results to find best parameter values.

Params to test:
-sigma: corresponds to the variance of RBF
-n_rbf: number of RBF functions
-alpha: regularization parameter corresponding to tightness of fit

A k-fold or LOO cross-validation would be a more rigorous way to find
the best parameter values, however, even with a single split of the
data, I have to check 3^4 models, which takes ~500 seconds to compute.

'''
cutoff=525887 #cutoff at year 2016

results=pd.DataFrame() #df for results

#specify parameter values to test
sigma1=[100,500,1000] 
sigma2=[100,500,1000]
alphas=[0.01,0.05,0.1,0.5,1,5,10]
center1s=[5,50,100]
center2s=[15,100,500]

#test centers and sigmas with alpha being held constant at 0.1
MSEs,sigma1s,sigma2s,centers1,centers2=[],[],[],[],[]
for i in sigma1:
    for j in sigma2:
        for k in center1s:
            for l in center2s:
                MSEs.append(regr(df,cutoff,center1(k),center2(l),i,j,0.1))
                sigma1s.append(i)
                sigma2s.append(j)
                centers1.append(k)
                centers2.append(l)
results["sigma1"]=sigma1s
results["sigma2"]=sigma2s
results["center1"]=centers1
results["center2"]=centers2
results["MSE Training Error"]=MSEs
